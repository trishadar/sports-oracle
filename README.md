# 🏆 Sports Oracle — AI Betting Intelligence

A full-stack sports betting prediction app powered by Claude AI with real-time web search. Features a dark, tactical UI with live signal generation, daily slate analysis, parlay evaluation, and news intelligence.

---

## ✨ Features

- **Signal Generator** — Analyze any bet with AI: spread, moneyline, totals, props
- **Daily Slate** — Scan today's games and surface the highest-value plays
- **Parlay Analyzer** — Multi-leg analysis with correlation risk assessment
- **News Intel** — Betting-relevant news and injury reports in real-time
- **Signal History** — Track your saved picks
- **Live WebSocket** — Real-time activity feed as Oracle analyzes

## 🤖 AI Agent Capabilities

Each signal is generated by Claude with web search access, analyzing:
- 📰 Latest injury reports and lineup news
- 📊 Recent team/player performance trends
- 💰 Line movement and sharp money indicators
- 🌡️ Weather, venue, and contextual factors
- 📈 Historical head-to-head matchup data
- ⚡ Breaking news that moves lines

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- An Anthropic API key

### 1. Clone / Open in VS Code
```
code sports-oracle
```

### 2. Install dependencies
```bash
npm run install:all
# or manually:
cd backend && npm install
cd ../frontend && npm install
```

### 3. Configure environment
```bash
cd backend
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
```

### 4. Run development servers
```bash
# From root (runs both simultaneously):
npm install  # installs concurrently
npm run dev

# Or separately:
npm run dev:backend   # http://localhost:3001
npm run dev:frontend  # http://localhost:5173
```

Open **http://localhost:5173** in your browser.

## 🛠️ Extending

### Add a database for signal history
Replace the in-memory store in `backend/src/routes/signals.js` with SQLite:
```bash
cd backend && npm install better-sqlite3
```

### Add user authentication
Add JWT auth middleware to protect API routes.

### Add more data sources
Extend `bettingAgent.js` to include additional web searches for:
- Vegas Insider / Action Network odds
- Advanced stats (FiveThirtyEight, ESPN)
- Weather APIs for outdoor sports
